WEB SECURITY

The purpose of web security is to prevent attacks of this (or any other) kind. More formally, security is the action/practice of protecting websites from unauthorized access, use, modification, destruction, or interruption.

HTTP vs HTTPS protocol

The difference between these two protocols goes beyond adding a letter at the end. This letter, it should be clarified, adds the quality of "safe" to the protocol. So, HTTPS is the acronym for Hypertext Transfer Protocol Secure. This letter "S" refers to an additional protocol called Secure Sockets Layer (SSL) that together with HTTP allows the transmission of encrypted information on the Internet.

Using the HTTPS protocol is the safest way to browse the internet, since all the information we enter will be encrypted, which gives us the guarantee that the only ones who can access that information are the client and the server.

HTTPS is used in activities that add the use of personal data, either in banking, emails or anywhere else where passwords or any confidential information must be entered

What is Tor?

Tor is a network that implements a technique called Onion Routing, designed with a view to protecting communications in the United States Navy. The idea is to change the traditional routing mode of the Internet to ensure anonymity and data privacy.

The traditional routing we use to connect to servers on the Internet is straightforward. For example, if you want to read a website your computer connects directly to its servers. The route is relatively simple: from your computer to your router, from there to your ISP's routers and then straight to the web servers you're visiting.

The bad thing is that if someone intercepts the data packets at an intermediate point, they will know perfectly well where they come from and where they are going. Even if the data for each packet is encrypted on the HTTPS page, the packet headers are not encrypted, and the sender and recipient fields (among others) are still visible.

How does it work?

Onion Routing consists of sending data through a non-direct path using different nodes. First, computer A, which wants to send the message to B, calculates a more or less random route to the destination passing through several intermediate nodes. It then obtains the public keys of all of them using a directory of nodes.

Using asymmetric encryption, computer A encrypts the message like an onion: layered. It will first encrypt the message with the public key of the last node in the route, so that only it can decrypt it. In addition to the message, it includes (also encrypted) directions to the destination, B. This entire packet, along with instructions for getting to the last node in the list, is encrypted again so that only the penultimate node in the path can decrypt it.

The process is repeated until we finish with all the nodes of the route. With this we already have the data package ready, so it's time to send it. Computer A connects to the first node of the route, and sends the packet to it. This node decrypts it, and follows the instructions it decrypted to send the rest of the packet to the next node. It will decrypt again and send back to the next one, and so on. The data will eventually reach the exit node, which will send the message to its destination.

The VPN

A VPN or virtual private network.

Is a tool that allows you to protect your connection while you browse the internet by creating a private network connection between devices. VPNs are used to transmit data securely and anonymously over public networks.

What are VPNs for?

VPN services are primarily used to send data securely over the Internet. The three main functions of VPNs are:

o Privacy: Encryption is used to maintain the privacy of our confidential information,
especially when connecting through public wifi networks.
o Anonymity: They hide the IP address so that it remains anonymous on the Internet.
o Security: They use cryptography to protect your Internet connection from unauthorized access.

How do VPNs work?

These connections redirect data packets from your machine to another remote server before sending them to third parties over the Internet. The main features of VPN technology are as follows

VPN Types:

These are

VPN Types: In case Architecture
• Remote access VPN: A remote access VPN allows you to use the Internet to connect to a private network securely by accessing content shared on the network through encryption.
• Site-to-site VPN connections: They are used to connect two or more networks to each other between different sites.
• Extranet-based site-to-site connections It is the same as the previous one, with the only difference that it is used to connect networks of customers, suppliers and external usiness partners.

VPN Types: By Price
• Free" VP of Labor: Workers who use it do not need to pay for it, as the company itself pays for it, and the user must connect to it through a VPN client.
• Free VPN: A free VPN allows you to test the service, but with certain limitations on its use.
• Paid VPN: If you want to enjoy better benefits, you have to pay for it, since watching certain content consumes a large amount of data that costs money, these VPNs are faster and have better functions.

VPN Types: By Services offered
• VPN for work: This is the classic VPN connection with which we connect to the office to access shared resources as if we were there.
• VPN for streaming: These are usually paid connections that allow us to connect to the Internet as if we were in another country and it is fast enough to use it to watch streaming content that is available there.
• VPN to bypass censorship or pretend to be on another site: With our device, we can connect to a server in another country and "pretend" to be there, but if we do not need top speed or a lot of data traffic, a free VPN that is not optimized for streaming may be sufficient.
• VPN to hide the IP: It is a simple VPN connection with which we seek to hide our IP address so as not to be tracked.
• VPN for enhanced privacy and security It is also common to use a VPN connection to encrypt our data so that it is not easy to intercept what we do on the Internet.

Differences between free and paid VPNs

Paid VPN services finance costs by charging users a subscription fee. Free VPN services make money at the user's expense.
